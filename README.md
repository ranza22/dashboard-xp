Dashboard de progresso com sistema de XP e level feito com HTML, CSS e JavaScript + cronômetro para estudar enquanto aprende as linguagens e registra seu tempo, sempre que reiniciar a página ira reserta o tempo mais seu progresso das linguagens não. Primeiro projeto.

# Dashboard XP

- Barras de progresso
- Cronômetro
- Salvamento com localStorage
